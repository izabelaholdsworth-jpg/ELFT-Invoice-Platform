# ELFT Invoice Platform

A comprehensive Finance Intelligence Portal for East London NHS Foundation Trust (ELFT) that provides real-time financial analytics, contract management, and AI-powered insights for procurement and spending operations.

![Database](https://img.shields.io/badge/Database-PostgreSQL-orange)
![AI](https://img.shields.io/badge/AI-Claude%20Sonnet%204-purple)

## Features

- Real-time KPI Dashboard
- Contract Intelligence Registry  
- AI-Powered Finance Assistant
- Power BI Integration
- Non-PO Spend Tracking

## Quick Start

```bash
# Clone and setup
git clone <repo-url>
cd ELFT_Invoice_Platform
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt

# Configure
cp .env.example .env
# Edit .env with your credentials

# Run
python app.py
```

Visit http://localhost:5000
